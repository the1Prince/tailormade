Add app icons and splash here:

- **icon.png** – 1024×1024 (app icon)
- **splash.png** – 1284×2778 or similar (splash screen)
- **adaptive-icon.png** – 1024×1024 (Android adaptive icon foreground)
- **notification-icon.png** – 96×96, white on transparent (Android notification icon)

Use the generated `tailormade-logo.png` (tape measure + TM, black & white) from the project root as reference, or export from design tools. Background: use `#0a0a0a` for splash and adaptive icon.
